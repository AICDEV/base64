# base64
Simple base64 encod and decode in C

## Compile
```bash
 gcc b64.c -o b64 -Wall -O3 
```

## Encode
```bash
 ./b64 "base64 is sometimes weird to implement but it makes fun"
 YmFzZTY0IGlzIHNvbWV0aW1lcyB3ZWlyZCB0byBpbXBsZW1lbnQgYnV0IGl0IG1ha2VzIGZ1bg== 
```

## Decode
```bash
 gcc b64.c -o b64 -Wall -O3 
```
